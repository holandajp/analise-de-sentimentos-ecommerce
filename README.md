## Análise de Sentimentos em E-commerce com NLP

Projeto de NLP para classificação de sentimentos em textos de e-commerce, utilizando pré-processamento textual, TF-IDF e N-grams para treinamento de modelos de Machine Learning.

## Objetivo

Classificar automaticamente o sentimento de avaliações e comentários de clientes, transformando dados textuais em informações úteis para apoio à tomada de decisão.

## Tecnologias

Python, Scikit-learn, Pandas, NumPy, NLTK.
